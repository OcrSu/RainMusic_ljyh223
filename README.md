# RainMusic

根据RainMusic 二次开发的网易云第三方，因为原作者已经没有在维护了，趁着暑假改了一下。



因为有些地方还不是很好，所以，还请自己打包。可以在设置中填入你的网易云cookie中的MUSIC_U字段，并退出重新进去即可。

如果有bug，欢迎提交issues。如果代码如不了眼，也可以来pr。欢迎~
